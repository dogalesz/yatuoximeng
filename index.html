Why I Love Low-Level Programming

Low level programming is more exciting to me than managing datasets in python or coding neural networks. I also find x86 syntax more logical than the one in some of the higher-level languages. Oh, and I can write code that runs on any machine, no premade libraries.

My Electronics Projects Journey

I started messing around with simple Arduino projects at 12, then I stopped for a while but at 13, I joined an electrical engineering course in my school, where I first learned analog circuits and simple microcontroller circuits with the ATmega16A. Later, we moved on to use the STM32 for various projects, which included using the STM CubeIDE. On the winter of 2024, I started coding in C, and then x86 assembly. I've made countless projects and I collected about 20000 or more electronics components.
